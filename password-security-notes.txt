Password Security Analyzer - Project Notes

Purpose:
This project is a browser-based cybersecurity portfolio app. It helps users understand why some passwords are weak and what makes stronger passwords harder to guess.

What it demonstrates:
- HTML structure
- CSS layout and responsive design
- JavaScript DOM interaction
- Password entropy estimation
- Common password and pattern detection
- Leetspeak and base-word normalization
- Attack-model crack-time estimates
- Sanitized security reporting
- Security-focused user feedback

Demo checklist:
1. Open index.html in Chrome.
2. Type "password123" and show the weak score.
3. Click "Patterned" and show how "Summer2026!" is flagged for common base words and year patterns.
4. Click "Passphrase" and compare the stronger result.
5. Click "Generate" and show a stronger password example.
6. Explain that the app runs locally in the browser and does not send passwords anywhere.

Security disclaimer:
This is an educational tool. Do not use real passwords in public demos.
